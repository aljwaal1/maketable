# صورة إلى Excel

تطبيق Android Flutter عربي لتحويل صورة فاتورة أو جدول إلى ملف Excel/CSV.

## الميزات
- اختيار صورة من المعرض أو التصوير بالكاميرا.
- OCR باستخدام Google ML Kit عند الطلب فقط، وليس عند فتح التطبيق.
- تحليل ذكي لفواتير الكاش حتى بدون جدول مرسوم.
- مراجعة وتعديل الصفوف قبل التصدير.
- تصدير Excel و CSV ومشاركتهما.

## البناء
ارفع المشروع إلى GitHub ثم شغّل GitHub Actions. سيظهر APK داخل:

`apk/excel-scanner-ar.apk`
