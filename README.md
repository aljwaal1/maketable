# صورة إلى Excel - Excel Scanner AR

تطبيق Flutter Android يحول صورة فاتورة/جدول إلى ملف Excel و CSV.

## المميزات
- تصوير فاتورة أو اختيار صورة من المعرض.
- OCR باستخدام Google ML Kit.
- تحليل ذكي لفواتير الكاش حتى بدون جدول مرسوم.
- تحويل الأرقام العربية/الهندية إلى إنجليزية.
- مراجعة وتعديل الخلايا قبل التصدير.
- تصدير XLSX و CSV ومشاركة الملف.

## البناء
ارفع المشروع إلى GitHub ثم شغل Actions. سيظهر APK داخل مجلد `apk`.
